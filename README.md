# 🔐 Network Scanner

![Banner](PASTE-YOUR-BANNER-LINK-HERE)

🚀 A fast and multi-threaded network scanning tool built in Python to detect open ports and running services on a target system.

---

## 📌 Description

This project is a **custom-built network scanner** designed to identify open ports on a given IP address or domain. It supports scanning across a user-defined port range (1–65535) and uses multi-threading to improve performance.

It demonstrates key cybersecurity concepts such as:

* 🔍 Reconnaissance
* 🌐 Port Scanning
* ⚙️ Service Enumeration

---

## 🚀 Features

* ✅ Scan IP address or domain
* ✅ Custom port range (1–65535)
* ✅ Multi-threaded (fast scanning ⚡)
* ✅ Detect open ports and services
* ✅ Error handling for invalid input
* ✅ Clean and readable output

---

## 🛠️ Tech Stack

* 🐍 Python
* 🔌 Socket Programming
* ⚡ Threading
